# Macro

域名：`macro`

子代理宏编排域，将多步工具调用组合为可复用的宏流程。

## Profile

- full

## 典型场景

- 多步反混淆流程
- 自动化分析管线
- 用户自定义宏

## 常见组合

- macro + core + transform

## 代表工具

- `run_macro` — 按 macroId 执行宏（多步工具链），支持输入覆盖和超时控制。
- `list_macros` — 列出所有可用宏（内置和用户自定义），包含名称、描述、标签和步骤数。
